Border-Top-Width

Set the width of the top border.

Syntax
      border-top-width: length | thin,  medium,  thick |inherit ;

length - A measurement in px, pt or em alternatively
thin - 1px
medium - 3px
thick - 5px
inherit - inherit the width from the parent element.

The border-style property must be set for the border-width or border-color properties to have any effect.

Examples:
h1 { border-top-width: 10px; }
.ss64class { border-top-width: thick; }
#SS64id { border-top-width: 2em; }
