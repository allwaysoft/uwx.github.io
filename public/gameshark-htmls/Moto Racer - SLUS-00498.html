Moto Racer - SLUS-00498

Always Place 1st

800A5516 0701
8008F4CC F9F1
With this code, after you load the codes turn the code off and then when the races starts turn the code on then flip it back off. Do this each time a race starts and you will have Lapped the opponents once and have a 63 second timer. If you don't flip the code off you will continue to race on Lap 2 forever.
