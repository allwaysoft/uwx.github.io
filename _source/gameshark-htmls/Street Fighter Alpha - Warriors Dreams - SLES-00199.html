Street Fighter Alpha - Warriors Dreams - SLES-00199

NOTE

Two sets of codes are provided because of the way that the game calculates it's bonus. This is done from the remaining time, energy, supercombos etc. Hence, using the first set will lead to enormous bonuses and a long delay while they are calculated. It is better to use the second set, which will still provide all the functions of the codes, but reduce the bonus time. You can use any value for the codes between the normal and low value codes for varying levels or bonus.
