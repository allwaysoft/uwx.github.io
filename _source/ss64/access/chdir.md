<!-- #BeginLibraryItem "/Library/head_access.lbi" --><!-- #EndLibraryItem --><h1>ChDir</h1>
<p>  Change the current directory or folder.</p>
<pre>Syntax
      ChDir <i>path</i>

Key
   <i>Path</i>   The directory to make current.
</pre>
<p>The ChDir statement can be used in VBA code. </p>
<p><b>Example</b></p>
<pre>ChDir "C:\work\"

</pre>
