<!-- #BeginLibraryItem "/Library/head_access.lbi" --><!-- #EndLibraryItem --><h1>Time</h1>
<p>  Return the current system time.</p>
<pre>Syntax
      Time

</pre>
<p>The Time() function can be used in VBA or in an <a href="syntax-functions.html">SQL query.</a> </p>
<p><b>Example</b></p>
<pre>dim dtmCreated
dtmCreated = Time</pre>
