IsEmpty

Return TRUE if the expression is a variable has been initialised, otherwise return FALSE.

Syntax
      IsEmpty(expression)

Key

expression    A variable.

The IsEmpty() function can be used in VBA.

Examples

Dim intDemo as Integer, result
result = IsEmpty(intDemo)
 Returns True
