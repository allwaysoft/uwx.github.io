<!-- #BeginLibraryItem "/Library/head_vb.lbi" --><!-- #EndLibraryItem --><h1>Chr</h1> 
<p>Function that returns the character represented by an <a href="../unicode.html">Ascii code</a>.</p>
<pre>Syntax
      Chr (<i>ascii_code</i>)</pre>
<p><span class="code">Chr (65)</span> will return A </p>
<p><br>
Example</p>
<p class="code">Dim strDemo<br> strDemo = Chr (65)<br>
&gt; A </p>
