<!-- #BeginLibraryItem "/Library/head_orapack.lbi" --><!-- #EndLibraryItem --><h1>PL/SQL Packages UTL_URL</h1> 
<p>Convert (escape) a string containing reserved characters into 
  a valid Uniform Resource Locator (URL).</p>
<pre>Functions:

ESCAPE Function 
          Returns a URL with illegal characters (and optionally reserved characters)
          escaped using the %2-digit-hex-code format 

UNESCAPE Function
          Unescapes the escape character sequences to their original forms in a URL.
          Convert the %XX escape character sequences to the original characters   </pre>
