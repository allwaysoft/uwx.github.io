<!-- #BeginLibraryItem "/Library/head_orapack.lbi" --><!-- #EndLibraryItem --><h1>PL/SQL Packages DBMS_LIBCACHE</h1> 
<p>Prepare the library cache on an Oracle instance by extracting 
  SQL and PL/SQL from a remote instance and compiling this SQL locally without 
  execution.</p>
<pre>Subprocedures:

COMPILE_CURSORS_FROM_REMOTE 
           Extract SQL in batch from the source instance and compile the SQL
           at the target instance.
