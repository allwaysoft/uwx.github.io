<!-- #BeginLibraryItem "/Library/head_orapack.lbi" --><!-- #EndLibraryItem --><h1>PL/SQL Packages DBMS_SHARED_POOL</h1> 
<p>Display the size of objects in the shared pool, and mark them 
  for keeping or unkeeping in order to reduce memory fragmentation. </p>
<pre>Subprocedures:

SIZES  Show objects in the shared pool that are larger than the specified size 

KEEP   Keep an object in the shared pool 

UNKEEP Unkeeps the named object 

ABORTED_REQUEST_THRESHOLD 
       Set the aborted request threshold for the shared pool</pre>
