<!-- #BeginLibraryItem "/Library/head_orapack.lbi" --><!-- #EndLibraryItem --><h1>PL/SQL Packages UTL_REF</h1> 
<p>Reference to an Object Table.<br>
  These procedures enable you to write generic type methods without knowing the 
  object table name. </p>
<pre>Subprocedures:

SELECT_OBJECT Select an object given a reference. 

LOCK_OBJECT   Lock an object given a reference. 

UPDATE_OBJECT Update an object given a reference. 

DELETE_OBJECT Delete an object given a reference. </pre>
