<!-- #BeginLibraryItem "/Library/head_access.lbi" --><!-- #EndLibraryItem --><h1>ChDrive</h1>
<p>  Change the current drive.</p>
<pre>Syntax
      ChDrive <i>drive</i>

Key
   <i>drive</i>   The drive letter to switch to.
</pre>
<p>The ChDrive statement can be used in VBA code. </p>
<p><b>Example</b></p>
<pre>ChDir "T"

</pre>
