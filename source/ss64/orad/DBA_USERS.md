   ___________________________

      Name of the user
   USER_ID
      ID number of the user
   PASSWORD
      Encrypted password
   ACCOUNT_STATUS
      --
   LOCK_DATE
      --
   EXPIRY_DATE
      --
   DEFAULT_TABLESPACE
