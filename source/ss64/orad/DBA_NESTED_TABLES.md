<pre>Columns
   ___________________________
 
   OWNER
      Owner of the nested table
   TABLE_NAME
      Name of the nested table
   TABLE_TYPE_OWNER

      Name of the type of the nested table
   PARENT_TABLE_NAME
      Name of the parent table containing the nested table

   STORAGE_SPEC

   RETURN_TYPE
