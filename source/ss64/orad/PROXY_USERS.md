<!-- #BeginLibraryItem "/Library/head_orad.lbi" --><!-- #EndLibraryItem --><h1>PROXY_USERS </h1><p> List of proxy users and the client on whose behalf they can act. </p> 
 
<pre>Columns
   ___________________________
 
   PROXY
      Name of a proxy user
   CLIENT
      Name of the client user who the proxy user can act as
   CREDENTIAL
      Credential passed by the proxy for the client (if any)
   VERSION
      Version of the credential
   MINOR
      Minor number of the credential
   FLAGS
      Flags associated with the proxy/client pair
