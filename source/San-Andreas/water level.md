This tool changes the water level and wave size. Run it before starting GTA:SA. The "Restore Defaults" button puts WATER.DAT and WATER1.DAT back to the original files. (water level = 0 is not the same as the defaults, because water would be anywhere that's below sea level) Use water level -1000 for no water.
