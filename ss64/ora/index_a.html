ALTER INDEX

Change the properties of an index.

Syntax:

   ALTER INDEX [schema.]index options

Options:
The options used with this command can be any combination of the following

    ENABLE
    DISABLE
    COALESCE
    UNUSABLE
    RENAME TO new_index_name
